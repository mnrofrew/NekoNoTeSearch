Neko no Te - Parallel Web Search Tool
Useage:
Press ctrl+L, type "neko" + space in the address bar, type search keyword, and hit enter. Then the browser will open multiple tabs and show search results from all search engines listed below
Options:
Feel free to modify the list below. %s will be the search keyword that gets sent to the tabs. If you put a # at the beggining of the line, that line would be ignored If you put a search URL that is not really a URL, the extension withh comment that out.


Notes:
The search engine URLs should be synchronized over cloud
